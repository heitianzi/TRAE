# 社稷归墟传 · 项目介绍

## 一、项目是什么

**社稷归墟传** 是一款以明朝开国历史为背景的 **2D 像素风 RPG 网页游戏**，基于 Python + Pygame 开发，并通过 WebAssembly（pygbag）编译为浏览器即开即玩的网页游戏，无需下载安装。

玩家将扮演朱元璋，从钟离县的一介布衣出发，历经皇觉寺出家、淮西流浪、投奔红巾军、鄱阳湖大战、北伐元大都，最终登基称帝，亲历"布衣天子"的传奇崛起之路。

- **项目名称**：社稷归墟传 · 大明王朝 RPG
- **游戏类型**：2D 像素风 · 历史 RPG · 开放世界
- **技术栈**：Python 3.12 / Pygame 2.x / pygbag (WASM) / JSON 数据驱动
- **运行方式**：浏览器在线游玩 / 本地 Python 运行 / 单文件 HTML 离线运行
- **在线试玩**：<https://heitianzi.github.io/TRAE/>

---

## 二、项目亮点

### 🌟 亮点 1：真实历史 × 开放世界

在 **120×120 的超大瓦片地图**上构建了一个以钟离县为枢纽、四通八达的十字路网世界，包含 **22 个历史名城**（襄阳、南昌、广州、杭州、应天府、北京、元大都、云南等），玩家可自由探索、不走线性剧情，每个地点都承载着真实的历史脉络。

### 🌟 亮点 2：浏览器即开即玩（零安装）

借助 **pygbag 将 Python 编译为 WebAssembly**，玩家无需安装任何环境，打开网页即可游玩。同时提供单文件 HTML 版本（约 400KB），可拷贝到手机、平板、电脑离线运行，真正做到"一份代码，多端运行"。

### 🌟 亮点 3：完整的多系统玩法

游戏并非简单 demo，而是集成了 **9 大核心系统**，形成完整的 RPG 闭环体验：

| 系统 | 亮点说明 |
|------|----------|
| 🏰 标题界面 | 状态机驱动 + 平滑过渡动画 |
| 🗺️ 开放世界 | 120×120 大地图 + 十字路网自由探索 |
| 👤 NPC 对话 | 10 位历史人物（道衍、徐达、宋濂等），每人 5 段独家对白 |
| 💰 交易系统 | 5 类商店（杂货/武器/防具/珍品/药铺）+ 36 件商品 + 买卖经济 |
| 📕 装备图鉴 | 5 级品质（白/绿/蓝/紫/橙）筛选 + 可视化展示 |
| 👘 皮肤系统 | 角色三阶段外观：农民 → 士兵 → 帝王红袍，随等级成长 |
| ⚔️ 战斗系统 | 回合制战斗 + 金币/经验奖励 + 等级成长 |
| 💾 存档系统 | 位置/等级/装备/金币/进度全量保存 |
| 🧭 小地图 | 实时小地图 + 世界地图 + 探索进度面板 |

### 🌟 亮点 4：数据驱动架构，易扩展

地图、装备、怪物、剧情、NPC、商店 **全部使用 JSON 配置**，与代码完全解耦。新增一个地点、一件装备、一段剧情，只需修改对应 JSON 文件，无需改动游戏逻辑——这种设计大幅降低了二次开发与内容创作的门槛。

### 🌟 亮点 5：代码生成美术资源

36 张地图瓦片图与全部角色/NPC 精灵图均由 **Python 脚本程序化生成**（`generate_player_sprites.py`、`generate_npc_sprites.py`），无需依赖外部美术资源或付费素材，体现了"代码即美术"的极客创意，也保证了项目的完全原创性。

### 🌟 亮点 6：文化价值——让历史"可玩可感"

游戏以《明史》为蓝本，涵盖 1344 年旱灾、皇觉寺出家、投奔红巾军、鄱阳湖大战、北伐元大都等真实事件，配以 10 位历史人物的多段对白与剧情文本，让玩家在游戏过程中自然地了解明初历史，实现**寓教于乐**的文化传播价值。

## 三、技术架构速览

```
┌─────────────────────────────────────┐
│           UI 层 (Game.draw_ui)       │
│   属性面板 / 装备栏 / 小地图 / 剧情   │
├─────────────────────────────────────┤
│         游戏逻辑层 (Game 类)         │
│   战斗 / 交易 / 存档 / 事件处理       │
├─────────────────────────────────────┤
│         实体层 (Player / Camera)     │
│   玩家状态 / 移动 / 属性计算          │
├─────────────────────────────────────┤
│         数据层 (JSON 文件)           │
│   地图 / 装备 / 怪物 / 剧情 / NPC    │
└─────────────────────────────────────┘
```

- **开发语言**：Python 3.12
- **游戏引擎**：Pygame 2.x（asyncio 异步架构，适配 WASM）
- **网页部署**：pygbag 编译为 WebAssembly
- **数据驱动**：6 份 JSON 配置全量游戏数据
- **像素美术**：36 张瓦片 + 代码生成的角色/NPC 精灵

## 四、一句话总结

> **社稷归墟传** 是一款用 Python 写出来、能在浏览器里直接玩的明朝历史 RPG。它把真实历史事件、开放世界探索、完整 RPG 系统与代码生成的像素美术熔于一炉，既是一个技术 Demo，也是一次让历史"可玩可感"的文化尝试。
