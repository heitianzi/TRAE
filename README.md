# 社稷归墟传 · 大明王朝 RPG

> 一款以明朝开国历史为背景、基于 Python + Pygame 开发并通过 WebAssembly 在浏览器中即开即玩的 2D 像素风 RPG 网页游戏。

---

## 🎮 立即游玩

### 在线试玩（GitHub Pages）

👉 **[立即游玩 →](https://heitianzi.github.io/TRAE/)**

> 首次打开需联网加载运行时（约 30–60 秒），之后浏览器缓存，可离线游玩。

### 本地运行

```bash
# 方式一：直接用 Python 启动本地服务器
python3 -m http.server 8000
# 然后浏览器访问 http://localhost:8000/index.html

# 方式二：用源码运行（需安装 pygame）
pip install pygame
python3 main.py
```

### 单文件版（拷贝到任意设备）

仓库根目录下的 [`index.html`](./index.html) 是自包含的单文件版本（约 400KB），把它拷贝到手机、平板、电脑，通过本地 HTTP 服务器即可游玩，无需安装任何依赖。

## 📖 游戏介绍

跟随朱元璋从皇觉寺小和尚、流浪乞丐，到红巾军士兵，最终登基称帝的传奇经历。在 120×120 的大地图上探索 22 个历史名城，与历史人物对话，征战沙场，经商交易，体验"布衣天子"的崛起之路。

### ✨ 核心功能

| 系统 | 说明 |
|------|------|
| 🏰 标题界面 | 带状态机（TITLE / PLAYING）和平滑过渡动画的启动界面 |
| 🗺️ 开放世界 | 120×120 超大地图，22 个历史地点，十字路网自由探索 |
| 👤 NPC 对话 | 10 位历史人物（道衍、徐达、宋濂等），每位 5 段独特对白 |
| 💰 交易系统 | 5 类商店（杂货/武器/防具/珍品/药铺），36 件商品，买卖经济 |
| 📕 装备图鉴 | 按品质（白/绿/蓝/紫/橙）筛选，36 件装备可视化展示 |
| 👘 皮肤系统 | 角色三阶段成长外观：农民 → 士兵 → 帝王红袍 |
| ⚔️ 战斗系统 | 回合制战斗，金币与经验奖励，等级成长 |
| 💾 存档系统 | 保存位置/等级/装备/金币/进度，随时继续 |
| 🧭 小地图 | 实时小地图 + 世界地图 + 探索进度面板 |

### 🕹️ 操作说明

| 按键 | 功能 |
|------|------|

| `T` | 与 NPC 对话 |
| `Space` | 推进对话 |

## 🛠️ 技术架构

- **开发语言**：Python 3.12
- **游戏引擎**：Pygame 2.x（asyncio 异步架构）
- **网页部署**：[pygbag](https://github.com/pygame-web/pygbag) 编译为 WebAssembly
- **数据驱动**：地图 / 装备 / 怪物 / 剧情 / NPC / 商店均使用 JSON 配置
- **像素美术**：36 张瓦片图 + Python 代码生成的角色 / NPC 精灵图

### 项目结构

```
game/
├── main.py                    # 游戏主程序（状态机、渲染、事件处理）
├── index.html                 # 单文件网页版（自包含，可直接部署）
├── maps.json                  # 地图数据（22 个历史地点）
├── equipment.json             # 36 件装备数据
├── npcs.json                  # 10 位 NPC 数据
├── monsters.json              # 怪物数据
├── stories.json               # 历史剧情文本
├── shop.json                  # 商店数据
├── sprite_peasant.png         # 农民皮肤（1-4 级）
├── sprite_soldier.png         # 士兵皮肤（5-9 级）
├── sprite_robe.png            # 帝王红袍皮肤（10 级+）
├── npc_*.png                  # 8 种 NPC 精灵图
├── tiles/                     # 36 张地图瓦片图
├── generate_player_sprites.py # 角色精灵生成脚本
├── generate_npc_sprites.py    # NPC 精灵生成脚本
├── build_standalone.py        # 单文件 HTML 构建脚本
└── sheji_guixu.html           # 创意展示页
```

### 重新构建网页版

```bash
# 1. 安装 pygbag
pip install pygbag

# 2. 构建 WASM
pygbag --build .

# 3. 生成自包含的单文件 index.html
python3 build_standalone.py
```
